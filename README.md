# Spicetify Installer
## Features
- **Spicetify Installation:** Quickly install Spicetify
- **Spicetify Upgrade** Quickly Update Spicetify      
# How to use 
> if you dont have spicetify Pick "Yes"

> if you have spicetify and it wont work pick "No"
